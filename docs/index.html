The Universal File Naming Convention

A great folder structure is only half the solution; a consistent file naming convention is what makes a system truly powerful and searchable.

The Golden Rule: Be consistent. The goal is a filename that tells you what the file is without even opening it.

The Anatomy of a Perfect Filename

The full, modular structure is:

[Date]_[Description]_[Version]_[Status].[ext]

You won't use every part for every file. You pick and choose the components that are relevant.

The System Explained

1. The Date Component: YYYY-MM-DD

This is the most important component for sorting. Always use the YYYY-MM-DD format to ensure files sort chronologically.

When to use a date?

Use a date when the file represents a snapshot in time (invoices, reports, meeting notes, photos, receipts, official documents).

Date Format Options:
  • Full Date (YYYY-MM-DD): For specific, day-related items.
    2025-11-21_MeetingNotes_ProjectAlpha.txt
  • Month (YYYY-MM): Perfect for monthly reports or bills.
    2025-10_ElectricityBill.pdf
  • Year (YYYY): For annual items.
    2025_AnnualTaxReturn.pdf

2. The Description Component: What_Is_This

This is the core identity of your file. Be descriptive but concise.

Rules:
  • Use underscores (_) or hyphens (-) instead of spaces.
  • You can use PascalCase (e.g., SalesSummary) if you prefer, but be consistent.
  • Good: HomeLoan_PreApproval_Letter
  • Bad: document 1 from bank

3. The Version Component: v1, v2.1

Use this for any file that will have multiple iterations. This avoids the dreaded Final_Report_final_FINAL_v2.docx.

Formats:
  • Simple: v1, v2, v3 for major versions.
  • Detailed: v1-1, v1-2 or v1.1, v1.2 for minor revisions.
  • Example: WebsiteMockup_v2.psd

4. The Status Component: draft, final

This clearly communicates the file's stage in its lifecycle. It's often used with the Version component.

Common Statuses:

draft, review, approved, final, published.

Example:
  • 2025-11-30_QuarterlyReport_v3_review.docx
  • 2025-12-05_QuarterlyReport_v4_final.pdf
